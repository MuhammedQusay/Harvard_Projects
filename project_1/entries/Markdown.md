# Markdown Explained

Markdown allows you to format text using simple syntax. Here's a demonstration of various elements:

## Basic Text Formatting

This is a paragraph (`This is just a line of text`).

This is a horizontal rule (`---`):

---

**This is strong text** (`**text**`).

*This is emphasized text* (`*text*`) or _italic text_ (`_text_`).

Here's some `code` (`` `code` ``).

## Headings

# Heading 1 (`# Heading 1`)
## Heading 2 (`## Heading 2`)
### Heading 3 (`### Heading 3`)
#### Heading 4 (`#### Heading 4`)
##### Heading 5 (`##### Heading 5`)
###### Heading 6 (`###### Heading 6`)

## Lists

### Unordered Lists

* Item 1 (`* Item 1`)
* Item 2 (`* Item 2`)
* Item 3 (`* Item 3`)

### Ordered Lists

1. First item (`1. First item`)
2. Second item (`2. Second item`)
3. Third item (`3. Third item`)

## Abbreviations

<abbr title="HyperText Markup Language">HTML</abbr> (`<abbr title="HyperText Markup Language">HTML</abbr>`).
